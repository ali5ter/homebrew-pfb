# homebrew-pfb

Homebrew tap for [pfb](https://github.com/ali5ter/pfb) — pretty feedback for Bash scripts.

## Install

```bash
brew tap ali5ter/pfb
brew install pfb
```

## Usage

After installing, source pfb in any Bash script:

```bash
source "$(brew --prefix)/lib/pfb/pfb.sh"
```

## Updating

```bash
brew update && brew upgrade pfb
```
